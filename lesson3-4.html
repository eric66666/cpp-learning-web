3-4 雙層迴圈

雙層迴圈

for(int i = 0; i < 5; i++){
  for(int j = 0; j < 4; j++){
    cout << "*";
  }
  cout << endl;
}

雙層迴圈意義上是迴圈裡面還有一層迴圈，此程式碼執行流程為：

i = 0 而且 i < 5 條件成立，進到第一層迴圈，j = 0 而且 j < 4 條件成立，進到第二層迴圈，輸出 * ，最後 j++ 。內層迴圈一直做到 j = 4 且 j < 4 條件不成立，跳出內層迴圈，輸出換行，最後 i++ 。

i = 1 而且 i < 5 條件成立，進到第一層迴圈，j = 0 而且 j < 4 條件成立，進到第二層迴圈，輸出 * ，最後 j++ 。內層迴圈一直做到 j = 4 且 j < 4 條件不成立，跳出內層迴圈，輸出換行，最後 i++ 。
