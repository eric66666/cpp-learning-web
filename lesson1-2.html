1-2變數宣告

Example

#include <iostream>
#include <string>
using namespace std;

int main() {
  int a = 1;
  float b = 2.33;
  char c = 'h';
  string d = "aaa";
  bool e = true;
  return 0;
}

第一個變數為int，這種變數型態叫做「整數」，可以存入整數，負數也可以。宣告方式如像上面Example的變數 a 其值為整數1。

第二個變數為float，這種變數型態叫做「浮點數」，可以存入有小數點的數字。宣告方式如像上面Example的變數 b 其值為2.33。

第三個變數為char，這種變數型態叫做「字元」，也就是文字，可以是英文、中文、數字...等等。字元的左右需要用單引號'包起來，宣告方式如像上面Example的變數 c 所示，其值為h。

第四個變數為string，這種變數型態叫做「字串」，也就是一串文字，可以是英文、中文、數字...等等。字元的左右需要用雙引號"包起來，在宣告前需要加上標頭檔#include <string> ，宣告方式如像上面Example的變數 d 所示，其值為aaa。

第五個變數為bool，這種變數型態叫做「布林變數」，其布林值只有兩種可能，true和false。true表示邏輯上的「真」，false表示邏輯上的「假」。 宣告方式如像上面Example的變數 e 所示，其值為true。布林值也可以用1和0表示，1為true，0為false
