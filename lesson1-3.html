1-3輸出輸入

Example 1

#include <iostream>
using namespace std;

int main() {
  int a = 1;
  int b;
  cin >> b;
  cout << b;
  cout << a << endl;
  cout << "Hello World!\n";
  return 0;
}

在C++中，輸入和輸出分別為cin和cout，以Example為例，先宣告一個整數型態 b ，cin >> b;這行的意思是在執行程式時，輸入的數字會存入變數 b 中， cout << b;  則會把變數b的值印出來。

cout << a << endl;，這行會輸出a的值，為1。後面的endl表示換行的意思。

cout << "Hello World\n!";，這行則是會輸出  Hello World，後面的"\n!"也是換行的意思，但他要寫在雙引號裡。

Example 2 - 基礎應用

#include <iostream>
using namespace std;
